# JAWS-UGとは

JAWS-UG（Japan AWS User Group）は、AWSユーザーが自主的に集まり、AWSに関する技術情報を交換し、ネットワークを構築するためのコミュニティです。

## JAWS-UG全体

- 2008年に設立
- 日本全国に支部がある
- AWSの公式ユーザーグループ
- 定期的に勉強会やハッカソンを開催

## JAWS-UG 長崎支部

長崎県を拠点に活動するJAWS-UGの支部です。

### 特徴

- **地域密着**: 長崎のエンジニアが集まる場所
- **初心者歓迎**: AWSを始めたばかりの方も大歓迎
- **実践的学習**: 実務で使える知識を共有
- **ネットワーキング**: 地域のエンジニアとのつながり

### 活動内容

- 毎月のLT会（ライトニングトーク）
- ハンズオン勉強会
- AWS認定試験対策
- 地域企業との交流

## 運営方針

- オープンで透明性の高い運営
- 参加者主体のコミュニティ
- 技術的な偏見を持たない
- 長崎の技術コミュニティの活性化に貢献
