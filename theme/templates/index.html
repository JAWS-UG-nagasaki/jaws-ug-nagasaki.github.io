コミュニティについて

JAWS-UG（Japan AWS User Group）長崎は、全国の AWS ユーザーコミュニティの長崎支部です。知識を共有し、技術的な課題に取り組み、つながりを構築する場所です。

Mission: 長崎でAWSを学び・楽しむ
Vision: 遊び・学ぶ場所を作る

経験豊富なアーキテクトであっても、S3 や Lambda の旅を始めたばかりであっても、初心者歓迎の雰囲気で実務で使える知識を共有します。
